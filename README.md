# Transkrypcje rozmów badawczych

Repozytorium zawiera pełne transkrypcje rozmów wykorzystanych w badaniu przeprowadzonym na potrzeby pracy magisterskiej pt. „Model konwersacyjnego doradztwa zakupowego opartego na dużych modelach językowych w procesie wyboru technologii konsumenckiej”.

Rozmowy zostały uporządkowane według scenariuszy testowych oraz modeli językowych. Materiały stanowią uzupełnienie rozdziału badawczego pracy i pozwalają odtworzyć przebieg badania.

## Zawartość

- `scenariusz-1/` – rozmowy dotyczące pierwszego scenariusza testowego
- `scenariusz-2/` – rozmowy dotyczące drugiego scenariusza testowego
- `scenariusz-3/` – rozmowy dotyczące trzeciego scenariusza testowego
- `scenariusz-4/` – rozmowy dotyczące czwartego scenariusza testowego

## Uwagi

Transkrypcje zostały udostępnione poza głównym tekstem pracy ze względu na ich objętość. Umieszczenie pełnych rozmów bezpośrednio w pracy znacząco zwiększyłoby jej długość i utrudniło czytelność części badawczej.
